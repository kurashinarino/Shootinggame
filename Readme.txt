【シューティングゲームのあそびかた】

■ どんなゲーム？

自分のキャラを動かして、敵を倒すゲーム

弾を撃ってスコアをためる

シンプルで楽しい！

■ ゲームの動かし方

Java が入ってるパソコンで遊べる

ターミナルやコマンドプロンプトで以下を入力

javac *.java

java Main

■ 操作方法

矢印キー：キャラを上下左右に動かす

Zキー：弾を撃つ

Pキー：一時停止（ポーズ）

Escキー：ゲームをやめる

■ 画面について

左上：スコアとライフ表示

ライフが0になるとゲームオーバー

敵を倒すとスコアが上がる

■ ゲームオーバーになる条件

ライフが全部なくなったら

Escキーを押したら

■ メモ

自分で少しずつコードを変えてみると、オリジナルゲームが作れる！

楽しみながらチャレンジしてみよう！
